Север для мобилки

## Quick Start

### 1. Клонирование репозитория
  
<pre>
git clone https://github.com/LasRom/ServerPart.git
cd ServerPart
</pre>  
  
### 2. Создание и активация окружения  
  
Windows  
<pre>
# Создание
python -m venv .venv

# Активация
.venv\Scripts\activate
</pre>  
  
Для Linux и macOS:  
  
<pre>
# Создание
python3 -m venv .venv

# Активация
source .venv/bin/activate
</pre>  
  
### 2. Создание и активация окружения  
  
<pre>
pip install --upgrade pip
pip install -r requirements.txt 
</pre>
